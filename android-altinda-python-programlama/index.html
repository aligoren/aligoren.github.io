Android Altında Python Programlama

26 Aug 2013

Merhaba bu yazımda size Android telefonlarınızda Python programlama yapma konusunda bi kaç bilgi vereyim. Oncelikle su konuda anlasalim bu yazi “python ile android uygulamaları geliştirme” hakkinda bilgi icermiyor. Buradan indirme islemini yapabilirsiniz. Program’in adi QPython.

BU PROGRAM PYTHON 2.7.X SURUMU ILE GELMISTIR. PYTHON 3.X DESTEKLEMEMEKTEDIR.

Linux, OS X ya da Windows’a kurduğunuz Python’u aynı şekilde Android sistemlere kurabilirsiniz. En dusuk sistemli telefonlar bile calistirabiliyor diye biliyorum. Eski telefonum olan Samsung Galaxy Ace’da testini yaptim. O telefondan ekran goruntulerini de atayim size

Oncelikle QEdit denen ekran’a bakacak olursak eger burada kodlarimizi yazip, calistirabilecegimiz ekrani gorebiliriz. Ortadaki ok tusu ile calistirabiliyoruz. <!--more-->

Daha sonra python konsolunda da calisma yapabiliyoruz. Evet konsol da var.

Son olarak da bazi linux komutlari calisiyormus onu fark ettim. Tamamini calistirabilmeniz icin root yetkisi gerekiyor. Ben ls komutunu calistirabildim.

Evet gordugunuz gibi istediginiz taktirde python programlama sansiniz da var. Tabletlerde kullanmak daha kolay olacaktir ama ben eski telefonumda bile kullanabildim gordugunuz gibi.
