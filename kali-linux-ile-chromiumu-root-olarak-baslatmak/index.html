Kali Linux ile Chromium’u Root Olarak Başlatmak

18 Jul 2013

Selam, Kali Linux Debian tabanlı bi Linux dağıtımıdır. Amacı Penetrasyon testleri yapmak. Yani güvenlik testleri diyebiliriz. Tabii chrome yani chromium kurup root olarak çalıştıramayanlar olabilir. leafpad metin editörü ile yüklü gelir onunla yaptığınızı var sayalım İsterseniz ilk işimiz chromium kurmak olsun. Bunu şöyle kuralım;

“leafpad /etc/apt/sources.list” yazıp, DEBİAN KAYNAK PAKET BAĞIMLILIKLARINI YAPMIŞ OLMANIZ GEREKMEKTE. Kaynak paket kodlarını alıp kendinizinki ile değiştirin. Sonra sırası ile apt-get update, apt-get upgrade yazın <!--more--> Kaynak kodlar üst kısımda sarı kutucuktaki kırmızı linktedir kurulumu yapalım;

apt-get install chromium-browser chromium-browser-l10n

Kurulum bitti şimdi açabilmek için gerekli ayarları verelim; Öncelikle chromium’u leafpad yardimi ile açalım. Aşağıdaki kod leafpad ile yapıldı. İstendiği taktirde gedit yüklenirse leafpad yerine onu da yapabilirsiniz.

leafpad /usr/bin/chromium

Bu şekilde yaptıktan sonra aramamız gereken bir satır var

exec $LIBDIR/$APPNAME $CHROMIUM_FLAGS "$@"

Bu satırı hemen aşağıdaki satır ile değiştirebilim ve kaydedelim.

exec $LIBDIR/$APPNAME $CHROMIUM_FLAGS "$@" --user-data-dir

İşlem bu kadar. Şimdi chromium browser’ı açabilirsiniz
